Next Code: A coding agent using Claude Max or ChatGPT Pro subscriptions

Usage: next-code [OPTIONS] [COMMAND]

Commands:
  serve                   Start the agent server (background daemon)
  acp                     Run as an Agent Client Protocol (ACP) adapter backed by the Next Code daemon
  server                  Manage the background server daemon (e.g. `next-code server stop`)
  connect                 Connect to a running server
  run                     Run a single message and exit
  login                   Login to a provider via OAuth, API key, or local credentials
  repl                    Run in simple REPL mode (no TUI)
  update                  Update next-code to the latest version
  version                 Show build/version information in human or JSON form
  usage                   Show usage limits for connected providers
  self-dev                Self-development mode: run as a canary session on the shared server
  debug                   Debug socket CLI - interact with running next-code server
  auth                    Authentication status and validation helpers
  provider                Provider discovery and selection helpers
  memory                  Memory management commands
  session                 Session management commands
  secrets                 Manage encrypted, keychain-backed secrets (API keys, tokens)
  ambient                 Ambient mode management
  permission              DCG permission mode management (allow-once codes)
  cloud                   Optional Next Code Cloud/Jade integration commands
  permissions             Review and respond to pending ambient permission requests
  transcript              Inject externally transcribed text into the active Next Code TUI
  dictate                 Run configured dictation: send to last-focused next-code client or type raw text
  setup-hotkey            Set up the platform global hotkey to launch next-code
  setup-launcher          Install a launcher so next-code appears in yo
